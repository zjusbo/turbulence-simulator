How it works
  • Domain is 40 × 24 ft; rulers show coordinates in feet.
  • Solver: lattice-Boltzmann (D2Q9) CFD with LES turbulence closure.
  • On Simulate, an adaptive quadtree mesh is generated and refined at object boundaries. Zooming in refines the mesh further. Enable the mesh view to see a wind arrow in every mesh cell.
  • Colors map local wind speed in MPH (blue = slow, red = fast). Flow accelerates in constrictions (venturi effect).
  • The rotor zone highlights cells with reversed (recirculating) flow behind objects; its total area is reported in ft².
  • When vortex shedding is detected the view shows an instantaneous single-frame field of direction & strength arrows.

    Finer mesh = more accurate CFD result, but longer precompute. The wind-arrow mesh is adaptive: the more complex the terrain (or the closer to objects), the finer the arrows.
    3D CFD (D3Q19 lattice-Boltzmann) cannot run in real time on a single machine at this resolution, so it is precomputed: the canvas locks while computing, then the recorded wind-arrow animation plays on a loop. Magenta arrows mark the rotor (recirculation) area.
